Este programa, basado en Java permite disponer del JSON oficial de la compañia Riot Games, y creo un programa que recoge todos los campeones de
este videojuego, permitiendo asi el conocimiento de todos su lore dentro del mismo y sus estadisticas, tambien se le añade un Quiz, un sencillo 
cuestionario de preguntas que salen aleatorias y por ultimo tiene un timer que permite llevar un control de las oleadas y los grandes monstruos
dentro del mismo
